****To set up and run a Gin and MongoDB project for a login system, you'll need to follow these steps **
1. Make sure you have Go installed on your system
2. Install Gin framework in VS Code.
3. For Backend I have used MongoDb database.
4. For the same make sure that MongoDB install on system.
5. Install MongoDB: Install MongoDB on your system. You can download it from the official MongoDB website: https://www.mongodb.com/try/download/community

6. Install MongoDB driver for Go: Install the MongoDB driver for Go using the following command: go get go.mongodb.org/mongo-driver
7 To perform the signup operation using Postman for the /auth/signup endpoint, you'll need to follow these steps:
  Open Postman: Open the Postman application on your system.
  Select POST Method: In the Postman interface, make sure the request method is set to POST.
  Enter Request URL: Enter the URL for the signup endpoint in the request URL field. For example: POST http://localhost:8080/auth/signup
8. Enter Request Body: In the text area provided, enter the JSON data for the signup request.
